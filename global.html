Members

constant heroes :Object

El siguiente objeto contiene toda la información acerca de los héroes con excepción de las url de las imagenes.
Example
{
        'id': 'dc-batman',
        'superhero':'Batman', 
        'publisher':'DC Comics', 
        'alter_ego':'Bruce Wayne',
        'first_appearance':'Detective Comics #27',
        'characters':'Bruce Wayne'
    }

constant heroImages :Array.<Object>

El siguiente Array contiene los url de cada héroe.
Example
{
     url: "https://res.cloudinary.com/heroe",
     name: (corresponde a la propiedad "id" en la data de "heroes.js")
}
